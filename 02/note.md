# Day 2

- Use math sorthand to manipulate value based on previous value

```python
score = 0

# shorthand
score += 1
print(score)
```

- Use `f-string` to print variable

```python
score = 10
height = 1.10
isActive = True

print(f"I've {score} score and height {height}. Am I active? {isActive}")
```
